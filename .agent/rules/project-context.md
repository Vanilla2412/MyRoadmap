# Project Overview
- **Project Name**: My Roadmap
- **Objective 1 (Self-Improvement)**: Master state-of-the-art to modern web technologies through AI-driven development.
- **Objective 2 (Market Value)**: Create a professional portfolio that demonstrates high-level engineering skills to global tech companies.

# Tech Stack
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: AWS Amplify Gen 2 (Infrastructure as Code via TypeScript)
- **Auth**: AWS Cognito
- **DB**: Amazon DynamoDB
- **CI/CD**: GitHub Actions

# Collaboration & Language Policy (Hybrid)
1. **Communication**: AIエージェントとの対話、複雑なロジックの相談、要件の深掘りは **日本語** で行います。
2. **Core Documentation**: `docs/` 内の要件定義（Requirements）やシステム設計（Architecture）は、正確を期すため **日本語と英語を併記** または **日本語を主** とします。
3. **Source Code**:
   - 変数名・関数名: **英語 (English)**
   - コード内コメント: **英語 (English)** ※グローバルな開発現場の作法に従います。
4. **Git Workflow**:
   - コミットメッセージ: **英語 (Conventional Commits準拠)**
   - プルリクエスト（PR）の説明: **英語 (English)**

# Development Guidelines
- 実装を開始する前に、必ず `docs/` フォルダ内の最新の要件定義を参照し、Plan（計画）を提示してください。
